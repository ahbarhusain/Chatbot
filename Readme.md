## Chatbot 

Ask anything related to our custom data

## Installation

Developed using `python 3.10` on windows.

```bash
pip install -r requirements.txt
```

## Token Key

```bash
Add your HUGGINGFACEHUB_API_TOKEN in key.py 
```

## Usage

```bash
chainlit run model.py
```
